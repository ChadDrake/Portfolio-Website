Fitnotes

FitNotes is an app I designed and developed to allow users to track their progress toward their fitness goals. Each user starts with the metrics of weight and body-fat percentage, and can add progress points to those metrics which will be displayed as a graph over time, so you can visualize your progress. Users can also add any other metrics they might want to. For example, their pull up count, or max bench press.

Spaced-Repetition

Spaced Repetition is a project I worked on in school. I designed and developed it after being given the project parameters. It's purpose is to help user's learn Spainish using the spaced repetition learning process. Users are given a word, and asked to guess the meaning. If they get it right, it is moved back in the list, if not, they will be asked again shortly.
